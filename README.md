WatchList Movie App is a modern Android application that allows users to explore movies, view detailed information, and save their favorites for later viewing.
It is built using Kotlin, follows MVVM architecture, and integrates the TMDB API to fetch real-time movie data.

‚≠ê Key Features
1.Browse Trending & Latest Movies

2.The app fetches trending, popular, and newly released movies directly from the TMDB API, ensuring users always see up-to-date content.

3.Powerful Movie Search

4.Users can search for any movie by its title and instantly view relevant results along with ratings and posters.

5.Detailed Movie Information

6.Each movie page displays complete details including overview, release date, rating, popularity, and poster/banner images.

7.Watchlist Management (Room Database)

8.Users can save their favorite movies to a personalized watchlist stored locally using Room Database, ensuring persistence even after app restarts.

9.Smooth & Modern UI

10.Built with Jetpack components and clean UI design for a smooth and intuitive user experience.
